## FAQ

**1) What’s the fastest way to connect?**  
Open the app → hit **Quick Connect** (or pick the nearest region). Don’t overthink it.

**2) When should I use Kill Switch?**  
Always. If the tunnel drops, Kill Switch stops traffic from leaking outside the VPN.

**3) How do I switch countries/regions?**  
Go to **Servers** → select a location from **Global Servers** → connect. The client will re-route instantly after handshake.

**4) How can I confirm the VPN is working?**  
Connect, then check your public IP and DNS behavior. If the IP changes to the selected region and DNS requests stay inside, you’re good.
