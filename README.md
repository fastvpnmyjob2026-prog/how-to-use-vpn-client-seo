## FAQ

### 1) How do I know the VPN is actually connected?
Open the client and check the status indicator. Then confirm your IP changed using an IP check site after connecting.

### 2) What should I enable first?
Enable **Kill Switch**. It prevents leaks if the connection drops.

### 3) Which server should I pick?
Start with **Quick Connect** for best speed. Switch regions only when you need a specific location.

### 4) Will this slow down my connection?
Any VPN adds some overhead, but with **high speed** routing and nearby servers, the hit is usually small.
