## FAQ

**Q: How do I confirm the VPN is actually working?**  
A: Connect, then check your public IP/location against your expected server region. Re-check after sleep/wake.

**Q: Should I leave Kill Switch on all the time?**  
A: Yes. If the tunnel drops, Kill Switch keeps traffic from falling back to your normal connection.

**Q: Which server should I pick for speed?**  
A: Start with the closest region. Switch only if you need a specific country or a service requires it.

**Q: Will this slow down my internet?**  
A: Any VPN adds overhead, but a nearby server with a stable connection should keep speed high.
